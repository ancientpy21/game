## Directions
1. Run the game by accessing the code
2. there will be a start screen, click the start
3. Use the `up` key to jump
4. avoid the enemy as long as you can
5. if you touch the enemy, there will appear a game over sign shows your score
6. you can start it back by clicking 'start game' again
